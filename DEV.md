# BeepBox 2.3.1 modding notes

This repo is set up to be easy to edit and test locally.

## One-time setup

- Install local dev tooling: `npm install`

## Dev loop

- Start dev mode (watch rebuilds): `npm run dev`
- In another terminal, serve the site: `cd website && python3 -m http.server 8000`
- Open: `http://localhost:8000/`
- Edit TypeScript in `editor/` and `synth/` and refresh the browser.

Notes:
- The editor page is `website/index.html`.
- The offline single-file build is `website/beepbox_offline.html` (generated by `npm run build`).

## Build outputs

Generated files live in `website/`:
- `beepbox_editor.js` / `beepbox_editor.min.js`
- `beepbox_synth.js` / `beepbox_synth.min.js`
- `beepbox_offline.html`
